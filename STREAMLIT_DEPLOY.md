# Deploy no Streamlit Cloud

## Passo a Passo

1. **Acesse**: https://share.streamlit.io/
2. **Faça login** com sua conta GitHub
3. **Clique em "New app"**
4. **Configure**:
   - **Repository**: `WilliamPrecs/logs`
   - **Branch**: `main`
   - **Main file path**: `app.py`
5. **Clique em "Deploy"**

## Executar Localmente

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Vantagens do Streamlit

- ✅ Deploy gratuito e ilimitado
- ✅ Atualização automática do GitHub
- ✅ Interface Python simples
- ✅ Sem limite de deploys
